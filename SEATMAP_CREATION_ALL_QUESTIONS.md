# Seat Map Creation - Complete Interview Questions

**Version**: 1.0  
**Date**: 2026-05-12  
**Scope**: All Questions for Seat Map Creation Discussion

---

## 1. TEMPLATE MANAGEMENT

### Library & Reuse
- Do you have a template library system for reusing seat map layouts across multiple events?
- How many templates do you typically store per venue?
- How do you handle template versioning when an event is already published?
- Can you roll back to a previous template version?

### Draft/Publish Workflow
- What is your draft/publish workflow for seat maps?
- Can you save a draft before going live?
- Who approves templates before they're published?

### Bulk Operations
- Can you import seat maps from external sources like CSV or Excel files?
- What's the maximum number of seats you can create in a single operation?

---

## 2. ZONE CONFIGURATION

### Zone Pricing
- How do you organize zones by price ranges?
- Can users navigate zones by price tier?
- Do you have VIP zones with special pricing?

### Zone Assignment
- Can a single seat belong to multiple zones for pricing purposes?
- How do you handle capacity synchronization between zones and actual seats?

### Zone Identification
- How do you visually distinguish VIP zones from standard seating areas?
- Is there a specific icon or color scheme for VIP zones?

---

## 3. ELEMENT CREATION

### Seat Generation Patterns
- Can you generate seats in curved or irregular patterns, or only in grid layouts?
- What percentage of your venues require non-grid layouts?

### Collision Detection
- How do you prevent overlapping elements when designing a seat map?
- What warnings do you get during design or errors after publishing?

### Bulk Creation
- What's the maximum number of seats you can create in a single operation?
- How long does it take to create 5,000 seats?

### Element Properties
- Can you rotate elements?
- How do you manage layer ordering (z-index)?
- Can you customize fill and stroke styles?

---

## 4. VALIDATION & QUALITY

### Layout Validation
- How do you validate that elements don't overlap during seat map creation?
- Can you place elements outside the canvas boundaries?
- How do you ensure zone capacity matches actual seats?

### Preview Mode
- Can you preview a seat map from a customer's perspective before publishing?
- What validation checks occur during this preview?

### Quality Assurance
- What quality checks are performed before an event goes live?
- Who performs these checks - automated system or manual review?

---

## 5. OPERATIONAL EFFICIENCY

### Setup Time
- How long does it typically take to set up a seat map for a 2,000-seat venue?
- How much of this time is spent on repetitive tasks that could be automated?

### Resource Utilization
- How many team members are typically involved in creating a new seat map?
- What's their skill level - technical or non-technical?

---

## 6. COST IMPACT

### Time Investment
- If you could reduce seat map creation time by 75%, how would that impact your operations?
- What would you do with the time saved?

### Error Costs
- How much time and money do you spend correcting seat map errors after publication?
- How often do these errors occur?

---

## 7. ROI-FOCUSED QUESTIONS

### Template Library Value
- If you had a template library that reduced setup time by 75%, what would that be worth to your business annually?
- Would you pay $50K for such a system?

### Validation System
- How much would you save annually if layout errors were reduced by 90%?
- Would you pay $40K for such a validation system?

### Bulk Operations
- What's the maximum number of seats you need to create in a single operation?
- Would you benefit from increasing the limit from 500 to 5,000?

---

## 8. RISK ASSESSMENT

### Layout Error Risk
- What's the business impact of a seat map with overlapping or incorrectly placed elements going live?
- Have you experienced this issue before?

### Capacity Mismatch Risk
- How do you ensure zone capacity matches the actual number of seats assigned?
- What happens if there's a mismatch?
